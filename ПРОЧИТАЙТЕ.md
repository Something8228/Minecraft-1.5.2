# Minecraft 1.5.2 Portable (No Mods)

Чистая сборка Minecraft 1.5.2 с установленным ModLoader. Работает без лаунчера.

## Особенности:
- **Portable**: Все данные (логи, настройки, сохранения) хранятся в папке с игрой.
- **Чистый клиент**: Никаких лишних модов, только база для их установки.

## Как запустить:
1. Скачайте архив из раздела **Releases**.
2. Распакуйте в любую папку.
3. Запустите файл `start.bat`.

## Как сменить ник:
1. Нажмите правой кнопкой на `start.bat` -> **Изменить**.
2. В строке `set NICKNAME=Player` замените `Player` на своё имя.
3. Сохраните и играйте!

**Требования:** Установленная **Java 8**.
